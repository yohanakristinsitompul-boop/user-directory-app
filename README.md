# User Directory App

## Deskripsi App

User Directory App adalah aplikasi React Native berbasis Expo yang mengambil data user dari REST API dan menampilkannya dalam bentuk list menggunakan FlatList.

## API yang Digunakan

https://jsonplaceholder.typicode.com/users

## Fitur

### Level 1 (Core)

* Fetch data dari REST API menggunakan async/await
* useEffect dengan dependency array []
* Loading State
* Success State
* Error State
* Retry Button
* FlatList
* renderItem
* keyExtractor

### Level 2 (Pengembangan)

* Search / Filter user berdasarkan nama
* Pull-to-Refresh

## Tech Stack

* React Native
* Expo Go
* JavaScript
* JSONPlaceholder API

## Cara Menjalankan

1. Clone repository
2. Install dependencies:
   npm install
3. Jalankan aplikasi:
   npx expo start
